# Mitgliedschafts-Challenge Spiel

Ein interaktives Spiel mit Supabase Score-Datenbank.

## Dateien

- `Spiel2.html` - Hauptspiel
- `score-datenbank.html` - Score-Datenbank-Anzeige
- `supabase-config.js` - Supabase-Konfiguration

## Netlify Deployment

1. Gehe zu https://app.netlify.com/drop
2. Ziehe den gesamten Ordner per Drag & Drop hinein
3. Fertig! Die Seite ist sofort online.

## GitHub Pages (Alternative)

1. Lade alle Dateien in dieses Repository hoch
2. Aktiviere GitHub Pages in den Repository-Settings
3. Wähle den `main` Branch als Source

## Supabase

Die Score-Datenbank verwendet Supabase Firestore für zentrale Datenspeicherung.
